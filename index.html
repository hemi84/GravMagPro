GravMagPro

A Python package for processing gravitational and magnetic data.

About

GravMagPro is a commercial Python software package tailored for the processing and analysis of gravitational and magnetic field data, specifically designed for oil & gas, mineral exploration, and other geophysical surveys. It offers an industry-standard suite of tools that streamline data processing and enhance decision-making in exploration projects.

Key Features

  • Basic Field Data Corrections: Includes automatic corrections for common issues in gravity and magnetic data, such as:
    • Instrument Drift: Compensates for temporal changes in instrument readings.
    • Base Station Correction: Ensures consistency by adjusting for base station readings for magnetic survey.
    • Latitude Correction: Adjusts for variations in the Earth's gravitational field with latitude.
    • Free-Air Correction: Accounts for elevation differences by adjusting gravity values.
    • Bouger Correction: Corrects for the gravitational effect of the terrain mass between the measurement point and sea level.
    • Terrain Correction: Further refines the Bouger correction by considering detailed terrain models.
  • Advanced Data Processing: Filtering, transforming, and enhancing gravity and magnetic data for deeper insights.
  • 3D and 2D Modeling & Inversion: Tools for generating and visualizing subsurface models for accurate exploration and resource estimation.
  • Anomaly Detection: Pinpoints and analyzes gravity and magnetic anomalies to help locate potential resource deposits.
  • Customizable Workflows: Modular pipelines that adapt to various exploration needs, from large-scale regional studies to detailed site analyses.

GravMagPro is available under a modular pricing system, allowing users to choose and pay for the features that align with their project requirements. This ensures flexibility and cost efficiency for projects ranging from basin-wide surveys to focused resource exploration.

Installation

To install gravmagpro, run the following command:

For local installation using a wheel package:

pip install /path/to/grav-mag.whl

For a .tar.gz distribution file:

pip install /path/to/grav-mag.tar.gz

Regular Updates

Users will need to download the latest distribution files and reinstall the package locally. The changelogs or update notifications will be provided via email or a secure communication channel.

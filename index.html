About the Platform

Precision Geophysics,
Programmable by Design

GravMagPro is a commercial Python software platform purpose-built for the processing, interpretation, and inversion of gravitational and magnetic field data. Developed by AICrux Business Solutions Pvt Ltd, it serves exploration geoscientists working in mineral discovery, oil & gas, and environmental surveys.

Unlike legacy GUI-based platforms, GravMagPro delivers the full power of a geophysical processing suite within Jupyter Notebook environments — enabling automated workflows, reproducible science, and seamless integration with modern AI and machine learning pipelines.

From basic field corrections through to GPU-accelerated 3D inversion and CNN-based target mapping, GravMagPro scales from individual consultants to large enterprise geoscience departments.

Platform Capabilities

A Complete Geophysical
Intelligence Toolchain

📐

Field Data Corrections

Diurnal, instrument drift, latitude, free-air, Bouguer, and terrain corrections — fully automated and parameterizable.

🌊

Advanced Signal Processing

FFT-based filtering, 1D/2D convolution, Butterworth, wavelet decomposition, and custom filter design for noise suppression and signal enhancement.

🔺

Derivative Transforms

Horizontal, vertical, and tilt derivatives; upward/downward continuation; total horizontal gradient and analytic signal computation.

🔍

Anomaly Detection & Euler Deconvolution

Automated source-depth estimation and structural index classification using standard Euler deconvolution algorithms.

🌐

3D Forward Modeling & Inversion

GPU-accelerated 3D gravity and magnetic inversion with drillhole integration, structurally constrained regularization, and multi-physics joint inversion.

🗻

Lineament & Structural Mapping

Automated detection of geological lineaments and porphyry-style structural corridors to identify mineralization-favorable zones.

🤖

AI-Driven Mineral Targeting

CNN-based anomaly classification and Bayesian prospectivity modeling for portfolio-level target ranking with explainable physics–AI fusion workflows.

📊

Data Gridding & Interpolation

Minimum curvature, kriging, and natural neighbor interpolation methods for survey data regularization and map generation.

Quick Start

Installation & Setup

GravMagPro is distributed as a Python wheel or source archive, delivered securely upon license issuance. Installation requires Python 3.8+ and a compatible Jupyter environment.

Wheel Package

Install from .whl

pip install /path/to/grav-mag.whl
Source Distribution

Install from .tar.gz

pip install /path/to/grav-mag.tar.gz

For GPU-accelerated inversion, CUDA 11.x or later is required. See the full Installation Guide for environment configuration and dependency details.

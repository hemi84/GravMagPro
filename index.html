About GravMagPro

GravMagPro is a commercial Python software package tailored for the processing and analysis of gravitational and magnetic field data, specifically designed for oil & gas, mineral exploration, and other geophysical surveys. It offers an industry-standard suite of tools that streamline data processing and enhance decision-making in exploration projects.

Key Features

  • Basic Field Data Corrections: Diurnal correction, instrument drift correction, lattitude correction, free-air correction, bouger correction, terrain correction
  • Advanced Data Processing: Filtering, transforming, and enhancing gravity and magnetic data for deeper insights.
  • 3D and 2D Modeling & Inversion: Tools for generating and visualizing subsurface models for accurate exploration and resource estimation.
  • Anomaly Detection: Pinpoints and analyzes gravity and magnetic anomalies to help locate potential resource deposits.
  • Customizable Workflows: Modular pipelines that adapt to various exploration needs, from large-scale regional studies to detailed site analyses.

Installation

To install gravmagpro, run the following command:

For local installation using a wheel package:

pip install /path/to/grav-mag.whl

For a .tar.gz distribution file:

pip install /path/to/grav-mag.tar.gz

For more detailed installation instructions, visit the Installation Guide.
